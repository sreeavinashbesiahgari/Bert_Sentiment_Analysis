Sentiment Analysis Project

This project implements a Sentiment Analysis application using the BERT model from the Hugging Face Transformers library. It allows users to input text, and the BERT model analyzes the sentiment of the text, returning a score ranging from Positive (1) to Negative (-1), indicating the sentiment polarity. The application is built using Streamlit for the user interface.

Features:
Input a sentence and analyze its sentiment.
Utilizes the BERT model for sentiment analysis.
Streamlit-based web app for easy interaction.

Steps to Run:

Clone the repository:


git clone https://github.com/sreeavinashbesiahgari/Bert_Sentiment_Analysis.git

Install the required dependencies:

pip install -r requirements.txt

Run the Streamlit app:


streamlit run app.py

Once the app is running, 

you can enter text and see the sentiment analysis results on the Streamlit UI.
